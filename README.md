# Proyecto Ecommerce (Monorepo)

Este es el inicio de un ecommerce completo + POS + sistema interno para reportes, auditoría, inventario y usuarios.

## Carpetas:
- apps/ → Frontend (Next.js)
- services/ → Microservicios (auth, productos, órdenes, auditoría, etc.)
- infra/ → Configuración de Docker y base de datos
- scripts/ → Scripts automatizados
- .github/ → CI/CD

Este proyecto se desarrollará por sprints, con enfoque profesional y escalable.
